Why do I need this?
Your browser stores translations in localStorage for faster loading. After updating translations in the database, you need to clear this cache to see the new translations.

Step 1: Clear localStorage

Step 2: Reload the page

What gets cleared:

  • translations_admin_he - Hebrew admin translations
  • translations_admin_en - English admin translations
  • translations_user_he - Hebrew user translations
  • translations_user_en - English user translations
Note: After clearing cache, you need to refresh the page for changes to take effect.

Manual Method (if buttons don't work):

Open DevTools Console (F12) and paste this:

localStorage.removeItem("translations_admin_he");
localStorage.removeItem("translations_admin_en");
localStorage.removeItem("translations_user_he");
localStorage.removeItem("translations_user_en");
location.reload();
